## 🧠 Fake News Detection using LSTM

Built and trained a **Long Short-Term Memory (LSTM)** neural network to detect fake news from a real-world news corpus. This solution can be practically implemented by media platforms to automatically identify whether a news article is fake, reducing the need for manual review across thousands of articles.
